Know What is Happening in Web Dev.

Web development is the work involved in developing a web site for the Internet or an intranet. Web development can range from developing a simple single static page of plain text to complex web-based internet applications, electronic businesses, and social network services.

What Does a Web Developer do.

A web developer can do various things in a project.

Coding Frontend Pages.

As a frontend developer you will be responsible in coding the designs for different pages(what the user sees).

Looking for Friendly Designs.

As web developer you may at times be responsible for looking for the design or themes to use.

Exploring Technologies.

As a developer part of the time you will be exploring various technologies that will help you in your work e.g wordpress,joomla. e.t.c

Some Technologies Being Used For Web Developent:

C.M.S

Content Management System (cms) is a piece of software where you do not need to go to code to change content. You also do not need to design, but you can pick a alreay made theme.

Bootstrap

Bootstrap is a technology that promotes mobile friendly pages. The screen is viewed as a series of rows and columns.

Drag and drop

Many softwares are now in the market that allows drag and drop to build a website.
Some of these softwares include: mobirise, wix e.t.c
